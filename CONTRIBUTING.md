# Contributing to NASA Jet Engine Predictive Maintenance

First off, thank you for considering contributing to this project!

## How to Contribute

### 1. Report Bugs
If you find a bug, please create an issue with detailed steps to reproduce it.

### 2. Suggest Enhancements
If you have ideas to improve the AI model or the dashboard, feel free to open a feature request.

### 3. Pull Requests
1. Fork the repository.
2. Create your feature branch: `git checkout -b feature/AmazingFeature`
3. Commit your changes: `git commit -m 'Add some AmazingFeature'`
4. Push to the branch: `git push origin feature/AmazingFeature`
5. Open a Pull Request.

## Technical Scope
- **Backend:** FastAPI, TensorFlow (LSTM)
- **Frontend:** Next.js 14, TailwindCSS, Recharts
